# Readme

## Controls

### Mouse

- **Left Mouse Button (LMB):** Drag cards - Use the left mouse button to click and drag a single card or a sequence of cards.
- **Double Click:** Auto-Move - Double-clicking on a card will automatically move it to the foundation if possible.

### Keyboard

- **Ctrl + Z:** Undo the last move.
- **Ctrl + H:** Show next hint.
- **Ctrl + C:** Collect all cards.
- **Ctrl + N:** Start new game.
- **Spacebar:** Deal new card.
